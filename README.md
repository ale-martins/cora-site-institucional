# 🫀 CoraApp — Site Institucional + Painel Administrativo

> Site institucional com painel administrativo para pesquisadores.



> 🔒 **Repositório privado** — este repositório contém apenas a documentação do projeto. O código-fonte é privado por questões de segurança e privacidade dos dados de pesquisa.



 🌐 **Website:** [saudecora.com.br](https://www.saudecora.com.br)

---

## Sobre

Este repositório contém o **site institucional** do CoraApp — apresentando o projeto, a equipe e os canais de contato — e o **painel administrativo** para os pesquisadores.

As aplicações web e mobile de acompanhamento de pacientes são repositórios separados.


---

## Stack tecnológica

### Core

| Tecnologia | Versão | Função |
|---|---|---|
| [Next.js](https://nextjs.org/) | 16.1.6 | Framework React com SSR/SSG e roteamento |
| [React](https://react.dev/) | 19.2.4 | Biblioteca de UI |
| [TypeScript](https://www.typescriptlang.org/) | 5.7.3 | Tipagem estática |

### Estilização

| Tecnologia | Versão | Função |
|---|---|---|
| [Tailwind CSS](https://tailwindcss.com/) | 4.2.0 | Utility-first CSS framework |
| [tailwind-merge](https://github.com/dcastil/tailwind-merge) | 3.3.1 | Merge de classes Tailwind sem conflito |
| [class-variance-authority](https://cva.style/) | 0.7.1 | Variantes de componentes tipadas |
| [clsx](https://github.com/lukeed/clsx) | 2.1.1 | Utilitário para classes condicionais |
| [tw-animate-css](https://github.com/jamiebuilds/tailwindcss-animate) | 1.3.3 | Animações CSS via Tailwind |

### Componentes UI

| Biblioteca | Função |
|---|---|
| [Radix UI](https://www.radix-ui.com/) | Componentes acessíveis e sem estilo |
| [shadcn/ui](https://ui.shadcn.com/) | Sistema de componentes sobre Radix UI + Tailwind |
| [Lucide React](https://lucide.dev/) | Ícones SVG |
| [Sonner](https://sonner.emilkowal.ski/) | Notificações toast (feedback das ações no painel admin) |

### Infraestrutura

| Serviço | Função |
|---|---|
| [Vercel](https://vercel.com/) | Hospedagem, deploy contínuo e CDN |
| [@vercel/analytics](https://vercel.com/analytics) | Analytics de uso |
